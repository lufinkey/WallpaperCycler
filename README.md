# WallpaperCycler
Cycles random wallpapers based on given keywords.

Work in progress. Right now it's built to run in Windows.
The goal is for it to eventually it will keep a database of already used wallpapers, and keep track of liked wallpapers.
Wallpapers are downloaded to %AppData%/Local/lufinkey/wallcycler/rotation and tags are stored in %AppData%/Local/lufinkey/wallcycler/tags.txt.
To set your wallpaper to rotate from the folder, set windows to rotate wallpapers from the rotation folder.
